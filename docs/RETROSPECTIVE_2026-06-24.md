## 0. 一句话总览

从"一个 LLM 聚类智能体(ACE)"出发，为冲 SCI 一区，依次尝试了 **通用聚类→癌症亚型(BRCA)→空间转录组(ST)** 三大方向、十余个具体子方向。
**结论：每个方向都被严谨地探到底——通用聚类是噱头/饱和；BRCA 分型饱和(打不过 PAM50)；ST 靶真实但三个创新探针全不亮。
个人从零"打赢成熟生信 SOTA 拿一区"这条路，已认真走完，未达成。** 但沉淀了可复用的流程、基线与方法论。

## 2. 各阶段关键发现（详）

### 2.1 原 ACE 审计
原项目是"多专家 LLM 现写 sklearn 代码→沙箱跑→按指标排名"。判据：去掉"agent/多代理"几个词系统行为不变→**是管线披 agent 皮，非真 agent**。做了 P0 修复 + 瘦身去重 + 剥离混入的心音子项目。

### 2.2 BRCA 亚型(核心负结果链)
- **稳定 ≠ 临床有用**(W6)：能造出统计上极稳定的亚型，但它们分不开病人生死；而成熟的 PAM50 能。原因：无监督抓"方差最大轴"(ER/Basal)，不是"最预后"的划分。
- **循环陷阱**(W7)：按生存挑基因又在同集测生存=自欺(训练 p 0.001 vs 留出 p 0.234)。必须留出验证。
- **多组学不是免费午餐**(W8)：朴素融合会淹没驱动焦点信号(ERBB2/Her2)。这是文献公认现象。
- **饱和**：连跨队列迁移(平台漂移)都被最朴素的最近质心解决(差距 0.02)。BRCA 分型是红海，无创新空间。

### 2.3 空间转录组(ST)
- **靶选对了**(W1)：与 BRCA 相反，ST 域识别上简单方法离 SOTA 有 0.28 的大空隙=真有提升空间。
- **基线立住**(W2)：复现官方 GraphST(SOTA)≈0.54 作对照。
- **三个创新探针全不亮**(W3)：多切片温吞；图像用 ImageNet 有害(编码器错)、换病理 FM(Phikon)做**公平的全分辨率测试**仍是**干净负**——因为皮层分层由表达梯度定义，H&E 形态在 spot 尺度几乎不含分层信息。

## 3. 横贯的方法学教训（这段时间最值钱的产出）

1. **先验证靶，再建方法**：用最简单基线测"离 SOTA 有多远"。远=有空间(ST)，近=饱和换靶(BRCA 跨队列只差 0.02)。这条纪律多次几小时就否掉了本要几周的方向。
2. **不把负/温吞结果包装成成功**：曾想把 BRCA 一串负结果包装成"认证框架"，被否——那是噱头。负就是负。
3. **稳定≠预后≠可复现**：三件不同的事，要分别验证。
4. **循环论证/留出验证**：挑特征和测效果不能用同一批数据。
5. **多模态要用对编码器 + 对的模态**：ImageNet≠病理；皮层分层该用表达不是图像；朴素拼接会让强势噪声淹没信号。
6. **inconclusive ≠ 负结果**：测试本身有缺陷(如数据损坏)时如实标注，别凑结论。
7. **数据可读性是真门槛**：HF 国内需镜像；Zenodo 的 DLPFC tif 截断损坏；高清病理图获取费劲——选数据先验证"能不能读/能不能用"。

## 5. 待决策的前路（给老师讨论）

冲一区的"正面硬打 SOTA"已尽力。更现实的几条：
1. **落一篇扎实二区/中游**：用现有 ST 资产做严谨的方法学/基准分析，或把多切片 +0.05 在强 backbone 上认真做成干净小改进+完整验证。可达性高。
2. **改打更纯 CS 优势的题**(不一定绑生信)：发挥 ML/工程长板，一区概率可能更高。
3. **找生物/临床合作者**：补领域短板，把"个人单干"变"有领域输入"，再冲生医一区。
4. **先厘清现实约束**：截止时间、可投入时长、目标是会议还是期刊、有无导师/合作——定了再选技术路线。

**关键现实**：CS 背景、无生物基础、单干，从零冲生信一区 SOTA 是极难的长跑(本项目即实证)。需要老师帮你校准"一区是硬指标还是理想"、以及是否引入合作/换题。
